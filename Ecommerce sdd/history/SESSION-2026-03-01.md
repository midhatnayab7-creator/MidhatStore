- `specs/002-shopping-cart/plan.md` — Full architecture plan
- `.specify/memory/agent-context.claude.md` — Agent context updated

**Key decisions**:
- Cart stored in Flask signed-cookie session (zero new dependencies)
- Prices locked at add-time as string (avoids Decimal JSON errors)
- PRG pattern (form POST → redirect, no AJAX)
- Context processor injects `cart_count` into all templates
